# 修改日志保存路径方案

## 当前结果位置

* **实验结果文件**：`/data/mzy/LibEER/result/few_channels_experiments/experiment_results.txt`

* **实验日志目录**：`/data/mzy/LibEER/log/few_channels_experiments/`

## 修改方案

### 1. 生成时间戳

在脚本开头添加时间戳变量，格式为`YYYYMMDD_HHMMSS`，用于创建唯一的日志目录。

### 2. 修改目录结构

将结果和日志保存到带时间戳的子目录中，例如：

* 结果目录：`/data/mzy/LibEER/result/few_channels_experiments/20251201_193000/`

* 日志目录：`/data/mzy/LibEER/log/few_channels_experiments/20251201_193000/`

### 3. 修改脚本内容

* 添加时间戳生成

* 修改目录变量

* 确保目录创建

### 4. 保留历史结果

每次运行都会生成新的目录，不会覆盖之前的结果，便于比较不同实验的结果。

## 修改步骤

1. 在脚本开头添加时间戳变量
2. 修改RESULT\_DIR和LOG\_DIR变量，加入时间戳
3. 修改目录创建命令
4. 测试修改后的脚本

## 预期效果

* 每次运行脚本都会生成带时间戳的目录

* 结果和日志保存在对应的时间戳目录中

* 便于管理和比较不同时间的实验结果

* 保留完整的实验历史

## 修改后的脚本结构

```bash
# 设置基本参数
BASE_DIR="/data/mzy/LibEER"
SCRIPT_DIR="$BASE_DIR/LibEER"
DATASET_PATH="/data/mzy/SEED/"

# 生成时间戳
TIMESTAMP=$(date +%Y%m%d_%H%M%S)

# 定义结果和日志目录，加入时间戳
RESULT_DIR="$BASE_DIR/result/few_channels_experiments/$TIMESTAMP"
LOG_DIR="$BASE_DIR/log/few_channels_experiments/$TIMESTAMP"

# 创建结果和日志目录
mkdir -p "$RESULT_DIR"
mkdir -p "$LOG_DIR"
```
